Final Year Project (FYP)

Respire Environmetal Data Air Quality System via Mobile Application

A mobile application named TACRES designed for asthmatic patients to evaluate their asthma by keeping track of the weather conditions and observation from tools as preventive measures to reduce the chances of asthma triggered. Flutter, dart, java and android studio were used in completing this project.

Login Page

The user may fill in the registered email and password to login the TACRES.

Home Page

Detect and display current location, weather and AQI, display current and previous exacerbation of one's asthma, and choice of taking GINA test or see record,

Sidebar Menu

It represents the sidebar menu of the TACRES application including GINA test, GINA record, FAQ, and language switches.

Weather Forecast

Display weather forecast hourly and daily for asthma users to predict the weather before they went outside.

GINA Questions

GINA is a preventive measure for asthma user. There are 4 listed questions and their answer field. The user only need to answer all question and click on submit button to get GINA Score.

GINA's Scores and Health Recommendation

The user that scores 0 which indicates that the asthma is well controlled, a score of 1-2 indicates that the asthma is partially controlled, and if the user scores 3-4 out of 4 means the asthma is out of control and the patients should seek help.

Below the score chart are health recommendations based on weather that appear as Asthma Action Plan (AAP). It display the current weather and suggest to the patient what is right

Page of GINA Data

It displays the history of the user while taking GINA. It display the date, the status of the GINA, the weather, the air quality index (AQI) of the place, and the place where the user took the GINA..

Asthma Action Plan Color Coded

The score result resulting with associated colour. If the user scores 0, the colour will display green as to show their asthma are well-controlled, while the score is between 1-2, the colour will be resulting in orange as it is Partly Controlled Asthma, and if the score is between 3-4, the colour will be red and the user need to seek help as the asthma is uncontrolled.
